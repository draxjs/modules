# Audit Vue

## Propósito

`audit-vue` aporta la capa visual del módulo: vistas de detalle, definición CRUD, dashboard y rutas para integrar auditoría en una aplicación Vue con Vuetify.

## Secciones

- `components`: widgets visuales para detalle y dashboard.
- `cruds`: metadata declarativa del CRUD de auditoría.
- `dashboards`: definición programática del tablero.
- `pages`: pantallas listas para router.
- `routes`: rutas que montan CRUD y dashboard.

## Cuándo usarlo

Usalo cuando quieras exponer auditoría en UI sin rearmar tablas, vistas y agrupaciones desde cero.
