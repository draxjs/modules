# AuditI18n

## Propósito

`Audit-i18n` define el vocabulario principal del módulo para `en` y `es`.

## Incluye

- nombre de entidad y menú
- labels de campos como `entity`, `user`, `action`, `changes`, `requestId` y `resourceId`
- permisos como `audit:view`, `audit:create`, `audit:update`, `audit:delete` y `audit:manage`

## Cuándo usarlo

Usalo como base de traducciones cuando montás CRUD, vistas o dashboard de auditoría.
