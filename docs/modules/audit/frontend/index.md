# Audit Frontend

## Propósito

`audit-front` encapsula el acceso cliente al módulo de auditoría y concentra sus traducciones reutilizables.

## Secciones

- `i18n`: textos para entidad, campos, acciones y permisos.
- `providers`: provider REST orientado a CRUD de solo lectura.

## Cuándo usarlo

Usalo cuando una app frontend necesite consultar auditoría y reutilizar el vocabulario del módulo.
