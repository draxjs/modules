# AuditPermissions

## Propósito

`AuditPermissions` define las claves de autorización del módulo.

## Valores

- `View = 'audit:view'`
- `ViewAll = 'audit:viewAll'`
- `Manage = 'audit:manage'`

## Uso típico

Se usa desde controladores, rutas y pantallas para limitar consulta y administración de auditoría.
