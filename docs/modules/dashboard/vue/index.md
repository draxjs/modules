# Dashboard Vue

## Propósito

`dashboard-vue` aporta la capa visual y declarativa del módulo: selector, render de tarjetas, editor de dashboards, rutas, store y CRUD.

## Secciones

- `combobox`: selector de dashboards existentes.
- `components`: visualización y edición de tarjetas.
- `composables`: lógica compartida para resolver entidades y cargar datos.
- `cruds`: definición declarativa del recurso.
- `pages`: pantallas listas para router.
- `routes`: rutas del módulo.
- `stores`: store Pinia para dashboards cargados.

## Cuándo usarlo

Usalo cuando quieras configurar dashboards dinámicos o renderizarlos en aplicaciones Vue con entidades CRUD registradas.
