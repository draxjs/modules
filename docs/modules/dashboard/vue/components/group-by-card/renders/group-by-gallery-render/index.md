# GroupByGalleryRender

## Propósito

`GroupByGalleryRender.vue` presenta resultados agrupados como una grilla de tarjetas.

## Qué hace

- calcula total y porcentaje por item
- compone labels a partir de los campos agrupados
- usa una paleta de colores de Vuetify para diferenciar tarjetas
- sirve como alternativa compacta a tabla o gráficos
