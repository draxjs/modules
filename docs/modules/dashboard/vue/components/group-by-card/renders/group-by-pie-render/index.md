# GroupByPieRender

## Propósito

`GroupByPieRender.vue` dibuja un gráfico tipo donut en canvas para resultados agrupados.

## Qué hace

- calcula porcentajes y colores por segmento
- arma labels compuestos desde los campos agrupados
- dibuja slices, etiquetas externas y total central
- redibuja al cambiar datos o tamaño de ventana
