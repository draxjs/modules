# Dashboard Frontend

## Propósito

`dashboard-front` encapsula el acceso cliente a dashboards y sus mensajes traducibles.

## Secciones

- `i18n`: textos del módulo y permisos.
- `providers`: provider REST para consumir `/api/dashboard`.

## Cuándo usarlo

Usalo cuando una aplicación necesite cargar o actualizar dashboards desde frontend.
