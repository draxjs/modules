# DashboardI18n

## Propósito

`Dashboard-i18n` define mensajes para `en` y `es`.

## Incluye

- nombre de entidad y menú
- labels de `identifier`, `title`, `cards`, `filters`, `layout`, `groupBy` y `paginate`
- permisos `dashboard:view`, `dashboard:create`, `dashboard:update`, `dashboard:delete` y `dashboard:manage`
