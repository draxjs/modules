# Media Vue

## Propósito

`media-vue` aporta la capa visual del módulo: campos de upload, vista enriquecida de archivos, combobox, CRUD y rutas.

## Secciones

- `comboboxes`: selectores de archivos existentes.
- `components`: campos y vistas reutilizables.
- `composables`: carpeta reservada para helpers Vue del módulo.
- `cruds`: definición declarativa del recurso archivo.
- `pages`: pantallas listas para router.
- `routes`: rutas exportadas por el paquete.

## Cuándo usarlo

Usalo cuando quieras integrar uploads y visualización de archivos en formularios o pantallas CRUD de Vue.
