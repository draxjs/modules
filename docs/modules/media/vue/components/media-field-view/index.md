# MediaFieldView

## Propósito

`MediaFieldView.vue` muestra una vista enriquecida de un archivo con detalles y preview.

## Qué renderiza

- icono por tipo de archivo
- tabs de detalles y vista previa
- metadata como tamaño, fechas, hits, expiración y tags
- preview de imagen o PDF
- acciones para abrir o descargar el archivo
