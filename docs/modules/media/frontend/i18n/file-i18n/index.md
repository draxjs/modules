# FileI18n

## Propósito

`File-i18n` define el vocabulario principal del módulo para `en` y `es`.

## Incluye

- nombre de entidad y menú
- labels de metadata como `filename`, `mimetype`, `size`, `hits` o `expiresAt`
- textos de vista como detalles, preview, descarga y apertura externa
- permisos CRUD del recurso `file`
