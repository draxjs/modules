# AutoSchema

## Propósito

`AutoSchema` es un ejemplo compacto de `IEntitySchema`.

## Qué muestra

- módulo `seguro`
- entidad `Auto`
- campo `marca` con validaciones simples
- campo `modelos` como `array.object`
- objeto anidado con `nombre` y `versiones`

## Valor práctico

Sirve para entender el formato mínimo de entrada antes de pasar a casos más completos como los schemas de `people` en `packages/zuite/zuite-arch`.
