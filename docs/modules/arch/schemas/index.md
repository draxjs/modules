# Schemas

## Propósito

La carpeta `schemas` contiene ejemplos de entrada para el generador.

## Piezas principales

- `AutoSchema`: ejemplo mínimo de una entidad con campo simple y array de objetos.

## Cómo leer esta carpeta

No representa artefactos generados, sino definiciones fuente que sirven para entender cómo debe declararse una entidad antes de pasarla a `ArchGenerator`.
