# Template Base

## Propósito

`Template.ts` es un esqueleto mínimo para nuevas plantillas. Expone la firma estándar de una función template que recibe `IEntitySchema` y devuelve un string.

## Cuándo sirve

Es útil cuando querés extender Arch con nuevos artefactos generados y mantener el mismo patrón usado por el resto del módulo.
