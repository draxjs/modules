# Templates Front

## Propósito

Las plantillas de `front` generan la capa cliente y la configuración visual necesaria para usar una entidad en aplicaciones Vue de Drax.

## Qué producen

- Provider REST.
- Clase `EntityCrud`.
- Componente CRUD y página wrapper.
- Combobox para referencias.
- Ruta Vue.
- Archivos i18n y sus índices por módulo.
