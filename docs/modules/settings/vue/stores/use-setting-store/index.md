# useSettingStore

## Propósito

`useSettingStore` guarda en memoria los settings cargados y su estado de carga.

## Qué expone

- estado `loading`
- estado `settings`
- getters `isSettingsLoading`, `getSettings`, `getSettingByKey`, `getSettingValueByKey`
- acciones `setLoading`, `setSettings`, `updateSetting`
