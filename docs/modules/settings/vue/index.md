# Settings Vue

## Propósito

`settings-vue` aporta la capa visual del módulo: formularios por tipo, páginas de edición y store reactivo de settings.

## Secciones

- `components`: widgets de configuración y edición.
- `composables`: lógica compartida de carga y update.
- `pages`: pantallas listas para router.
- `routes`: rutas del módulo.
- `stores`: store Pinia de settings.

## Cuándo usarlo

Usalo cuando quieras exponer un panel de configuración agrupado por categorías dentro de una app Vue.
