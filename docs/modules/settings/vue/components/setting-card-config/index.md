# SettingCardConfig

## Propósito

`SettingCardConfig.vue` renderiza settings agrupados en cards, una por variable.

## Qué hace

- carga settings agrupados por categoría
- muestra `label`, `description`, visibilidad y permisos
- usa `SettingField` para renderizar el valor
- abre `SettingEditor` para edición puntual
