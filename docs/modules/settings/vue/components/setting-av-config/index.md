# SettingAvConfig

## Propósito

`SettingAvConfig.vue` muestra un panel administrativo amplio con búsqueda, agrupación por categoría y modales de visualización/edición.

## Qué hace

- carga settings al montar
- filtra por `key` y `description`
- muestra tipo, visibilidad, permiso y última actualización
- abre modales para ver descripciones y editar valores
