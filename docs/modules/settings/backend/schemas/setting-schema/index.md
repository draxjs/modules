# SettingSchema

## Propósito

`settingSchema` valida la definición base de una variable de configuración.

## Campos tipados

- `key`, `label`, `description`, `category`
- `type` con los tipos soportados por el módulo
- `regex`, `entity`, `entityValue`, `entityText`
- `prefix`, `suffix`, `permission`, `public`
- `updatedAt`, `updatedBy`
