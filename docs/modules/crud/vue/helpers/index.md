# Helpers

## Propósito

`helpers` reúne utilidades pequeñas de soporte para la UI CRUD.

## Piezas principales

- `getItemId(item)`: resuelve el identificador utilizable de un registro, incluso cuando la entidad no usa explícitamente la misma propiedad en todos los contextos.

## Cuándo usarlo

Sirve cuando componentes y composables necesitan una forma uniforme de identificar el item actual.
