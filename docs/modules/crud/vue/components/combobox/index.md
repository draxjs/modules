# Combobox

## Propósito

La subcarpeta `combobox` concentra selectores especializados reutilizables. Hoy la carpeta expone un único componente.

## Piezas incluidas

- [EntityCombobox](/modules/crud/vue/components/combobox/entity-combobox/)

## Cuándo usarlo

Usala cuando necesitás elegir dinámicamente una entidad CRUD registrada en `useEntityStore()`.
