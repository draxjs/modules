# Crud Frontend

## Propósito

`crud-front` encapsula providers REST reutilizables para consumir endpoints CRUD desde cliente.

## Secciones

- `providers`: bases REST para CRUD genérico y helpers de serialización de filtros.

## Cuándo usarlo

Usalo cuando una entidad frontend quiera exponer operaciones CRUD comunes sin repetir armado de URLs, query params y exportaciones.
