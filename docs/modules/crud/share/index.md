# Crud Share

## Propósito

`crud-share` concentra los tipos compartidos que usan backend, frontend y Vue para describir operaciones CRUD, metadata de entidades y opciones de UI.

## Secciones

- `interfaces`: contratos de provider, servicio, repositorio, filtros, exportación y definición de entidades.

## Cuándo usarlo

Usalo como fuente única de tipos para mantener alineadas las tres capas del stack CRUD.
