# Routes

## Propósito

La carpeta contiene arreglos de rutas Vue Router listos para registrar.

## Piezas principales

- `IdentityAuthRoutes`: login, recovery, registro y cambio de contraseña.
- `IdentityCrudRoutes`: páginas CRUD de usuarios, roles, tenants, sesiones y fallos.
- `IdentityRoutes`: composición general del módulo.
- `UserSessionCrudRoute`
- `UserLoginFailCrudRoute`

## Cuándo usarlo

Conviene cuando querés incorporar rápido las rutas del módulo sin reescribir la definición manualmente.
