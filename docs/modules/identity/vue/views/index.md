# Views

## Propósito

`views` contiene componentes de detalle y visualización de entidades de identidad.

## Piezas principales

- `UserView`
- `RoleView`
- `TenantView`
- `UserApiKeyView`

## Cuándo usarlo

Conviene en tablas, drawers o diálogos donde solo querés renderizar la entidad ya cargada sin mezclarla con edición.
