# i18n

## Propósito

La carpeta agrega mensajes específicos de la capa Vue para entidades que tienen UI administrativa propia.

## Piezas principales

- `UserSession-i18n`
- `UserLoginFail-i18n`

## Cuándo usarlo

Usalo cuando la aplicación Vue necesite etiquetas y textos específicos de vistas CRUD de sesiones o fallos de login.
