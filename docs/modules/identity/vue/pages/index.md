# Pages

## Propósito

`pages` expone páginas completas listas para integrar en el router de una app Vue.

## Piezas principales

- Auth y perfil:
  - `LoginPage`
  - `ProfilePage`
  - `PasswordChangePage`
  - `PasswordRecoveryRequestPage`
  - `PasswordRecoveryCompletePage`
  - `RegistrationPage`
- CRUD:
  - `UserCrudPage`
  - `RoleCrudPage`
  - `TenantCrudPage`
  - `UserApiKeyCrudPage`
  - `UserSessionCrudPage`
  - `UserLoginFailCrudPage`

## Cuándo usarlo

Usalo cuando quieras exponer pantallas funcionales del módulo con muy poco wiring adicional.
