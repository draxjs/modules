# Identity Vue

## Propósito

`identity-vue` ofrece una capa de UI reutilizable para autenticación, perfil y administración de identidad en aplicaciones Vue.

## Secciones

- `assets`: estilos y recursos visuales base.
- `combobox`: selectores de tenants, roles y permisos.
- `components`: widgets y formularios de auth/perfil.
- `composables`: acceso reactivo a systems de identidad.
- `cruds`: definiciones y formularios CRUD listos para usar.
- `i18n`: mensajes específicos de vistas Vue.
- `icons`: iconografía del módulo.
- `pages`: páginas completas de login, perfil y CRUD.
- `routes`: rutas prearmadas para auth y administración.
- `stores`: estado global con Pinia.
- `views`: vistas de detalle para entidades.

## Cuándo usarlo

Usalo cuando quieras integrar rápidamente identidad en una app Vue manteniendo consistencia con el stack Drax.
