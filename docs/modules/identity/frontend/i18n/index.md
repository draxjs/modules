# i18n

## Propósito

La carpeta compone mensajes de traducción para todo el dominio identidad.

## Piezas principales

- `identity-auth-i18n`
- `identity-user-i18n`
- `identity-role-i18n`
- `identity-tenant-i18n`
- `identity-userApiKey-i18n`
- `identity-userSession-i18n`
- `identity-userLoginFail-i18n`
- `identity-validation-i18n`
- `identity-permissions-i18n`
- `identity-audit-i18n`
- `IdentityI18nMessages` en `index.ts`

## Cuándo usarlo

Usalo para registrar en una sola vez las claves de texto necesarias para login, perfil y CRUDs de identidad.
