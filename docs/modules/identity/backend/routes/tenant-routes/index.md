# TenantRoutes

## Propósito

`TenantRoutes` registra las rutas CRUD completas de tenants.

## Qué registra

- `search`
- `findById`
- `all`
- `find`
- `findOne`
- `findBy`
- `findOneBy`
- `paginate`
- `create`
- `update`
- `delete`
- `export`

## Documentación

Usa `CrudSchemaBuilder(TenantSchema, TenantBaseSchema, TenantBaseSchema, 'tenant', 'openapi-3.0', ['Identity'])`.
