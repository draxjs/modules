# HTML

## Propósito

La carpeta contiene plantillas HTML simples usadas en respuestas o notificaciones del flujo de identidad.

## Piezas principales

- `RegistrationCompleteHtml`: template utilizado al completar verificaciones de registro.

## Cuándo usarlo

Sirve cuando el módulo necesita devolver una respuesta visual mínima en lugar de JSON, por ejemplo al confirmar un código recibido por email.
