# IEmailLayout

## Propósito

`IEmailLayout` tipa las opciones visuales del correo HTML.

## Campos

- `bodyStyle`, `maxWidth`
- `headerCentered`, `headerBgColor`, `headerColor`
- `headerImage`, `headerImageStyle`
- `headerTitle`, `headerTitleStyle`
- `headerLogo`, `headerLogoStyle`
- `footerBgColor`, `footerCopyright`, `footerContent`, `footerUnsubscribe`
