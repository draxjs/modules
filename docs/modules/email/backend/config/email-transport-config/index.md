# EmailTransportConfig

## Propósito

`EmailTransportConfig` enumera las claves de entorno del transporte de correo.

## Claves incluidas

- `EMAIL_TYPE`
- `EMAIL_SERVICE`
- `EMAIL_HOST`
- `EMAIL_PORT`
- `EMAIL_AUTH_TYPE`
- `EMAIL_AUTH_USERNAME`
- `EMAIL_AUTH_PASSWORD`
- `EMAIL_SECURE`
- `EMAIL_IGNORE_TLS`
- `EMAIL_RATE_DELTA`
- `EMAIL_RATE_LIMIT`
