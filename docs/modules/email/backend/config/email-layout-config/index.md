# EmailLayoutConfig

## Propósito

`EmailLayoutConfig` enumera las claves de configuración del layout HTML del correo.

## Claves incluidas

- `EMAIL_BODYSTYLE`
- `EMAIL_MAXWIDTH`
- `EMAIL_HEADER_CENTERED`
- `EMAIL_HEADER_BGCOLOR`
- `EMAIL_HEADER_COLOR`
- `EMAIL_HEADER_IMAGE`
- `EMAIL_HEADER_IMAGE_STYLE`
- `EMAIL_HEADER_TITLE`
- `EMAIL_HEADER_TITLE_STYLE`
- `EMAIL_HEADER_LOGO`
- `EMAIL_HEADER_LOGO_STYLE`
- `EMAIL_FOOTER_BGCOLOR`
- `EMAIL_FOOTER_COPYRIGHT`
- `EMAIL_FOOTER_CONTENT`
- `EMAIL_FOOTER_UNSUBSCRIBE`
