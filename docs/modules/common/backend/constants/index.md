# Constants

## Propósito

`constants` agrupa valores compartidos del backend para evitar strings mágicos repetidos.

## Piezas principales

- `COMMON`, exportado desde `CommonConstants.ts`, actúa como punto central para constantes públicas del módulo.

## Cuándo usarlo

Usalo cuando otro paquete necesite referenciar valores comunes del backend sin duplicarlos.
