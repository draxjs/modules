# Common Vue

## Propósito

`common-vue` reúne piezas reutilizables para aplicaciones Vue basadas en Drax.

## Secciones

- `components`: componentes visuales apoyados en contratos compartidos.
- `composables`: hooks para validación, menús, fechas y portapapeles.

## Cuándo usarlo

Conviene cuando varios módulos Vue necesitan la misma base de UI y utilidades de interacción.
