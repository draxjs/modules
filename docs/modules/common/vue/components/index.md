# Components

## Propósito

La carpeta ofrece componentes listos para construir navegación, previews e inputs reutilizables.

## Piezas principales

- `SidebarMenu` renderiza un menú lateral desde `IMenuItem[]`, filtrando por permisos y autenticación con `useMenu`.
- `GalleryMenu` presenta variantes visuales de menú para vistas tipo galería.
- `MenuCard` encapsula items navegables en formato tarjeta.
- `IconCombobox` y `ColorCombobox` simplifican selección de iconos y colores.
- `DraxImagePreview` muestra imágenes con acciones como apertura o descarga.
- `DraxInfoItem` renderiza pares etiqueta/valor de forma consistente.

## Cuándo usarlo

Usalo para acelerar pantallas administrativas o catálogos donde el patrón de navegación y visualización se repite.
