# Common Share

## Propósito

`common-share` expone tipos mínimos que pueden ser consumidos tanto por frontend como por Vue sin arrastrar dependencias de runtime.

## Secciones

- `interfaces`: contratos para menús y formato de fechas.

## Cuándo usarlo

Es el punto correcto para publicar tipos compartidos entre paquetes cuando no hace falta lógica ejecutable.
